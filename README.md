# De_an_TN

## Mô tả dự án

Đây là dự án frontend cho hệ thống quản lý giáo dục, bao gồm các chức năng dành cho quản trị viên (admin) và giáo viên (teacher). Dự án được xây dựng bằng HTML, CSS và JavaScript.

## Tính năng chính

### Admin
- Course Management, Student Management, Class Allocation, Teaching Management

### Teacher
- Home, Income, Your Classes


## Công nghệ sử dụng

- **HTML5**: Cấu trúc trang web
- **CSS3**: Styling và responsive design
- **JavaScript (ES6+)**: Tương tác và logic phía client

## Cách chạy dự án

1. Clone repository này về máy:
   ```
   git clone https://github.com/LHAnhThu/De_an_TN.git
   ```

2. Mở file `login.html` trong trình duyệt web (Chrome, Firefox, Edge, v.v.)

3. Dự án sẽ chạy trực tiếp trong trình duyệt mà không cần server backend.
